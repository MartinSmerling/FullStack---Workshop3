# Workshop 03 – Express.js Web‑palvelin

Tämän projektin tarkoitus on näyttää, miten Express.js‑kehystä käytetään web‑palvelimen rakentamiseen. Express helpottaa huomattavasti Node.js:n sisäänrakennetun HTTP‑palvelimen käyttöä ja tekee reitityksestä, staattisista tiedostoista ja API‑päätepisteistä selkeitä ja helppoja.
